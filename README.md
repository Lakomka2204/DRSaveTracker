# DR Save Tracker
## Program for tracking DELTARUNE saves and backup them.

The issue this program resolves is steam cloud saves, they do not support uploading files while the game is running, so I made a program that will save ALL your save files.

When you're saving the game, the program will automatically backup the save to another folder where you can restore it's from when needed.

Most applicable use case scenario: "Oh fuck i forgot to claim that one item" or "Oh fuck i didn't do the [ACTION] during boss fight to check how will it react".

No AI was used during development, only for consultation (how to create an array).
